usage: vfxcli [-h] {summary,to-gltf,to-vfx,verify-roundtrip} ...

positional arguments:
  {summary,to-gltf,to-vfx,verify-roundtrip}
    summary             Print info about a VFX file
    to-gltf             Convert VFX to glTF (currently uses existing
                        vfx2obj.exe backend)
    to-vfx              Convert glTF to VFX (coming next)
    verify-roundtrip    Read->write and compare (lossless mode; coming next)

options:
  -h, --help            show this help message and exit
